# Example: Motif Sequence — Ascending Thirds

## Input

Motif: D4–F4–A4 (quarter notes)
sequence_spec: { transpositions: [0, 4, 7], direction: up, count: 3 }

## Parameters

strict_diatonic: false (chromatic transposition allowed)

## Output

Sequence:
1. D4–F4–A4 (original)
2. F#4–A4–C#5 (+4 semitones)
3. A4–C#5–E5 (+7 semitones)

## Notes

Ascending sequence by major thirds. Creates a rising arc. In D minor or F# minor context, the sequence outlines harmonic movement. The third restatement reaches A4–C#5–E5, a clear arrival. Useful for building to a climax or section transition.
