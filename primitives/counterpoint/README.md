# Counterpoint Primitives

Foundational primitives for contrapuntal writing. Used by Bach, Jimmy Wyble, Ellington counterlines, and hybrid engines.

- **voice_independence** — Two or more voices as independent melodic entities
- **contrary_motion** — Voices move in opposite directions
- **oblique_motion** — One voice sustains while another moves
- **parallel_avoidance** — Avoid parallel fifths and octaves
- **suspension_chain** — Prepared suspensions resolving downward
