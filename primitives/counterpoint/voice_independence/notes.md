# Notes: Voice Independence

## Rationale

Counterpoint requires voices that stand alone as melodies. Wyble's two-line guitar work and Bach's inventions exemplify: each line is singable, neither is mere accompaniment.

## Design Decisions

- Rhythmic independence is enforced: avoid identical attack patterns.
- Voice crossing is allowed when melodic logic is preserved.
- Harmonic meaning emerges from interaction, not from one voice "supporting" another.
