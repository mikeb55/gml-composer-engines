# Example: Counterpoint in B Section Only

**Engine:** Bach Counterpoint. **Section:** 32-bar AABA. **Window spec:** start_bar 8, end_bar 16, phrase_relative false.

**Result:** Counterpoint engine active only bars 8–15 (B section). A sections (0–7, 16–23) and final A (24–31) have no counterpoint output. Conductor invokes engine only for bars 8–15.

**Use case:** Arrangement where counterline appears only in bridge.
