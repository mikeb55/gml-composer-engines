# Example: Delay Resolution — Dominant Extension

## Input

harmonic_context: Cm7 | F7 | BbMaj7 (ii–V–I in Bb)
target_resolution: BbMaj7
delay_spec: { strategy: insertion, insertion_chord: "G7", extension_bars: 1 }

## Parameters

max_delay: 2

## Output

Progression: Cm7 | F7 | G7 | F7 | BbMaj7

## Notes

G7 (V/V) is inserted before the final resolution. The F7 is extended; G7 creates a secondary dominant that resolves to F7, which then resolves to Bb. The resolution is delayed by one bar. Tension increases before release. Classic jazz reharmonisation.
