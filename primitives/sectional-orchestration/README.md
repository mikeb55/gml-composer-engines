# Sectional Orchestration Primitives

Primitives for assigning roles, contrast, background figures, and density in big-band sectional writing. Core to the Ellington Orchestration Engine.

- **section_role_assignment** — Assigns functional roles (lead, pad, background, counterline, punctuation, support) to sections.
- **brass_reed_contrast** — Defines contrast behaviour between brass and reeds (colour, weight, response).
- **background_figure** — Generates supporting rhythmic/harmonic figures without stealing focus.
- **density_orchestration_map** — Controls orchestration density across phrases and sections.
