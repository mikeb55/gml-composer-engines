# Example: Playability Constraint — Two-Line Fragment

## Input

Two voices for solo guitar.

## Output

Upper voice: E4 F4 G4 A4
Lower voice: C3 D3 E3 F3

## Notes

Intervals (E–C, F–D, G–E, A–F) are playable within one position. Adjacent string movement (e.g. E string to A string) favoured. The primitive supports playable two-line motion by avoiding impossible stretches and excessive position shifts.
