# Example: String Set Preference — Two-Line Fragment

## Input

Two voices for solo guitar.

## Output

Upper voice: E4 F4 G4 A4 (A string, 5th–8th fret)
Lower voice: C3 D3 E3 F3 (D string, 5th–8th fret)

## Notes

Both voices on adjacent strings (A and D). Efficient string grouping; minimal picking hand movement. The primitive supports playable two-line motion by favouring adjacent string pairs and avoiding frequent large skips. Ergonomic picking patterns maintained.
