# Example: Register Layer — Melody and Background

## Input

material: melody (C4–G5), background figure (G3–D5)
layer_spec: { bands: [low, mid, high], assignment: { melody: high, background: mid } }

## Parameters

ensemble_spec: big band (trumpets, trombones, saxes, rhythm)

## Output

Register assignment:
- Melody → high band (C5–G5): lead trumpet or alto
- Background → mid band (G3–D5): 2nd/3rd trumpets or saxes

## Notes

Standard big band layout. Melody sits above the staff for clarity. Background occupies the mid register, leaving low for bass and possibly trombones. Avoids register clash. Ellington-style separation.
