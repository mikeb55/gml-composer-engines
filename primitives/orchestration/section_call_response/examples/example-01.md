# Example: Section Call Response — Brass and Reeds

## Input

material_a: 2-bar brass figure (staccato hits)
material_b: 2-bar reed figure (sustained)
section_spec: { call_group: brass, response_group: reeds, pattern: alternating, overlap: false }

## Parameters

alternation_pattern: [call, response, call, response] (8 bars total)

## Output

Timeline:
- Bars 1–2: Brass (call)
- Bars 3–4: Reeds (response)
- Bars 5–6: Brass (call)
- Bars 7–8: Reeds (response)

## Notes

Classic brass-reed dialogue. Brass states the figure; reeds answer. The contrast in articulation (staccato vs. sustained) reinforces the call-response. Builds momentum toward a tutti or solo section. Ellington-style section writing.
