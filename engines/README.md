# Engines

Style-specific engines built from primitives. Each engine combines primitives according to its decision logic and style constraints.

Categories: composer (Wayne Shorter, Andrew Hill, Bach, Ellington, Stravinsky), guitar (Jimmy Wyble).

**Jimmy Wyble Engine** — Two-line contrapuntal guitar engine based on Wyble's improvisational approach. Combines counterpoint primitives (voice independence, contrary/oblique motion, parallel avoidance) with guitar primitives (register partition, playability, dyad density, string set preference). Generates playable etude material with implied harmony.

Each engine has spec.yaml, notes.md, examples, and tests.
