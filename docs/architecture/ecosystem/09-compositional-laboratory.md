# Compositional Laboratory Framework

**Version:** 0.1.0  
**Layer:** gml-composer-engines (Layer 1)

---

## Purpose

Allow shared primitives and intelligence layers to be tested automatically across many scenarios. Usable by future engines.

---

## Framework Capabilities

| Capability | Description |
|------------|-------------|
| multi_candidate_generation | Generate many candidates; compare |
| scenario_based_primitive_testing | Test primitives across scenarios |
| motif_behaviour_logging | Log motif development behaviour |
| energy_curve_logging | Log energy arc behaviour |
| clarity_score_comparison | Compare clarity across candidates |
| novelty_memorability_comparison | Compare novelty and memorability |
| regression_checks | When primitives change, run regression |

---

## Laboratory Output

- Score distributions (average, best, worst)
- Primitive activation impact
- Repair effectiveness
- Export success rate
- Regression flags
